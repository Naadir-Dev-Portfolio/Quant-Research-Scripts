# Quant-Research-Scripts

Quantitative finance experiments — NeuralProphet time-series forecasting and inflation-adjusted analysis

Built by [Naadir](https://github.com/Naadir-Dev-Portfolio)

## Overview

Cutting-edge quantitative research combining NeuralProphet neural network models for time-series forecasting with inflation-adjusted stock price analysis. Explores the intersection of machine learning and fundamental financial analysis.

## Features

- NeuralProphet neural network time-series models
- Stock price forecasting with confidence intervals
- Inflation adjustment and real returns analysis
- Multi-asset backtesting
- Technical indicator integration
- Historical price analysis with macroeconomic context

## Tech Stack

Python · NeuralProphet · pandas · yfinance

## Setup

```
pip install -r requirements.txt
python main.py
```
